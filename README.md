小説「永遠のゆく先へ」のリポジトリです。

# About
「永遠の行く先へ」は、宇宙や地球のさまざまな場所時代で繰り広げられる少女たちの冒険を描いたSF小説です。この作品は、簡単にフォークできるようにしています。もし展開や演出に気に入らない部分があれば、あなた自身が修正して公開することが可能です。
